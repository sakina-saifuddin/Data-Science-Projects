1. Introduction:

a. Context:

This dataset contains information about the highest-grossing movies over a period of time. It includes details such as the rank of the movie, its title, the studio that produced it, the total gross revenue earned, and the year of release.

b. Objectives:

The objective of this dataset is to analyze the movies that have earned the most at the box office. It aims to identify trends in the film industry, such as the popularity of certain studios, years with higher earnings, and the impact of movie franchises on revenue.

c. Data Description, a short description of the column names:

i. Rank:

This column represents the position of each movie based on its gross earnings. The movies are ranked from the highest-grossing to the lowest within the dataset.

ii. Title:

The title column contains the name of the movie.

iii. Studio:

This column lists the studio responsible for producing the movie. Examples include Buena Vista, Fox, Universal, and others.

iv. Gross:

Gross represents the total earnings of the movie at the box office, displayed in millions of dollars.

v. Year:

This column indicates the release year of the movie.

Observation

  • The dataset tracks the highest-grossing movies over time.
  • It includes columns for Rank, Title, Studio, Gross, and Year.
  • Rank orders movies from highest to lowest gross earnings.
  • Title lists the name of each movie.
  • Studio identifies the studio responsible for the movie (e.g., Buena Vista, Fox, Universal).
  • Gross shows the total box office earnings in millions of dollars.
  • Year indicates the release year of the movie.

The dataset helps analyze trends in movie earnings, studio performance, and the impact of release years on box office success.

vi. Handling dimensionality reduction, if needed

Steps to Apply PCA:

Standardize the Data: PCA works better when all the features are on the same scale (e.g., if one feature has values in thousands and another in ones, PCA will be biased towards the larger numbers). So, we standardize the data first.

Apply PCA:

After standardizing, PCA will find the new features (principal components).

In[22]:
import pandas as pd
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler


# Step 1: Standardize the data
scaler = StandardScaler()
scaled_data = scaler.fit_transform(movies_copy[['Gross', 'Year']] )

# Step 2: Apply PCA (reduce to 2 components)
pca = PCA(n_components=2)
print("pca", pca)
pca_result = pca.fit_transform(scaled_data)
print("pca_result", pca_result)

# Add the PCA results to the DataFrame
movies_copy['PCA1'] = pca_result[:, 0]
movies_copy['PCA2'] = pca_result[:, 1]

print(movies_copy) # PCA is like creating new, smarter columns from your existing data.
